This README contains instructions to execute the SpamFilter.py file.

Following are the steps:



1. Execute this command : python Spamfilter.py train test1. Here "train" is the training data file and "test1" is the original test data file shared in the project.



2. Execute this command : python Spamfilter.py train test2. Here "train" is the training data file and "test2" is the additional test data to test the project.



3. Execute this command : python Spamfilter.py train test3. Here "train" is the training data file and "test3" is the additional test data to test the project.



4. The report is shared in the Report.pdf file for each of the command executed.
